Figma - Mobile application development

The idea was to develop a new user interface application - using the Figma program.

The idea was to develop a new user interface application - using the Figma program. The Shared APP is a user interface application intended for the management of various joint holdings. By using the application, the management of joint ownership is smooth and transparent, as all the necessary property-related information is available to everyone in real-time in one place, and handling joint matters becomes simpler. The application has the following features: homepage, login, main page, calendar, chat, to-do list, invoices, law, budget, photo album and ideas.
